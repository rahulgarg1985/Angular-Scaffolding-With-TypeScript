Angular-scaffolding-with-typescript

View project on GitHub

Overview

This repository contains an extensible scaffolding of angular with typescript. It has gulp based build and run configuration for dev and prod environment. It has following main components.

  • bower.josn : It contains all required javascript libraries for application.
  • package.json : It contains all required node packages for building and running the app.
  • tsd.json : There is a github repo, which hosts typescript definitions for a lot of javascript libraries. We can put informations of all required definitions in this file and they will be added to app. It is needed because we are writing our code in TypeScript.
  • gulpfile.json: This is gulp based configuration. It do all required stuff regaridng building and running the app.
  • config : This folder contains two files. 1. dev.js : This contains all external files (js and css) for dev environment. 2. prod.js : This contains all external files required for prod environment. Usually, we have location of minified files.
  • src : This folder have all application specific code in TypeScript. I divided this into two sub folder :
    • app : This folder contains app specific items, such as controllers, templates , directive, routes etc. Folder contains some basic code templates so that it would be work as reference.
    • core : I tried to abstract out some part of code, which would be quite similar for same flavour of applications, so that we can re use the same code base for different apps. For e.g.: Authorization,routing, storage, network calls, app level events, event-bus etc.

How to run

Go to root folder of Scaffolding and run below commands in order to install all required components and run.

  • npm install tsd bower gulp -g
  • npm install
  • bower install
  • tsd update
  • tsd rebundle
  • gulp
